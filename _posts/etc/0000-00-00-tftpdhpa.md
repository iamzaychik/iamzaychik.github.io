-----

1. Install:
```bash
$ apt install tftpd-hpa
```
2. Configure:
```
$ cat /etc/default/tftpd-hpa
TFTP_USERNAME="tftp"
TFTP_DIRECTORY="/path/to/firmware"
TFTP_ADDRESS="0.0.0.0:69"
TFTP_OPTIONS="--secure --create --umask 022 --permissive"
```
