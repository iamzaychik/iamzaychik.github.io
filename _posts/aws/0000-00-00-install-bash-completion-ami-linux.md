```bash
## Install bash-completion on Amazon Linux
$ wget http://www.caliban.org/files/redhat/RPMS/noarch/bash-completion-20060301-1.noarch.rpm
$ rpm -ivh bash-completion-20060301-1.noarch.rpm
$ . /etc/bash_completion
```
