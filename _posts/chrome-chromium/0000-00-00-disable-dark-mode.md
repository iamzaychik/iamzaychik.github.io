---
title: Disable dark mode
category: Chrome, Chromium
date: 2019-05-07
---

-----

**Add to shortcut:**
```bash
--disable-features=DarkMode
```

-----

```bash
Putting "chrome://flags" in the address bar (omit the quotes), then type "dark" in the Search box near the top of the page you see, to find the Dark mode dropdown box where there are multiple choices
```
