category: Ubuntu

-----

* Install (JDK or Java Development Kit, full version):
```bash
$ sudo apt install openjdk-8-jdk
```

* Install JRE (JRE or Java Runtime Environment, only executing):
```bash
$ sudo apt install openjdk-8-jre
```

* Check version:
```bash
$ java -version
```
